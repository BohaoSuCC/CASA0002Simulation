How to calculate Travel time for any location in the world

A guide on how to perform Isochrones (travel time from one point to another) in Python.

Different modes of transportation — i.e., cycling, walking, or driving — require different arrival times. But have you thought about, “Where can you reach 10 minutes of walking from your current position to any direction?”

That is the essence of Isochrones (travel time map). Isochrones ( in Greek isos means ‘equal’ and chrónos means ‘time’), refers to the line of equal travel time.

For example, a delivery company can use Isochrones to determine delivery areas that are reachable within 30 minutes drive time. Alternatively, find a new home that is ideal from your workplace using maximum commute time and transport mode preferences.

In this tutorial, I will show you how you can perform Isochrones (Travel time maps) within Python. We will use OSMnx Python library for street networks. The code for this article is available as Mybinder Notebook. Run and experiment it on the web without installing anything.

If you want to run locally on your computer, you need to install these three libraries with pip: networkx, Osmnx, and Geopandas.

Let us start with building street networks first.

Network Graphs

First of all, we need to get the data for the location we want. So, let us download first and visualize street network data for a place. In this tutorial, I will use Stockholm, Sweden, as an example. Feel free to pick up any other locations.

First, let us import the libraries.
